# loisa-composio-cli

Small Composio CLI for existing Tool Router sessions.

## Install

Run without installing:

```sh
npx loisa-composio-cli --help
```

Install globally:

```sh
npm install -g loisa-composio-cli
composio --help
```

The package name is `loisa-composio-cli`; the installed binary is `composio`.

## Development

```sh
npm ci
npm test
npm run typecheck
npm run build
```
